#CRM Data Cleaning & validation project 

##Overview
This project demonstrate my ability to clean, validate and prepare customer data using microsoft excel. The objective was to improve data quality, consistency and accuracy to produce a structured dataset suitable for reporting and business use.

##Business Problem
The dataset contained several data quality issues includint;
- Missing values and blank cells
- Inconsistent gender enteries(Male,M, female, F)
- Misspelled values(e.g Femal)
- Invalid entries (e.g U)
- Incorrectly formatted data values
- Inconsistent/Non-Standardised Job industry category values (e.g "Agriculturee", "n/a")
-Blank cells across multiple columns 

These issues reduced the accuracy, consistency and reliability of the dataset for reporting and analysis.

##My Respnsibilities

- Reviwed the dataset to identify data quaality issues.
- Standardised inconsistent gender values.
- Corrected missplled values and invalid enteries.
- Corrected formatting issues, including data vakues. 
- Identified and handeled missing values.
- Improved data consistency and quality.
- Prepared the dataset for reporting and analysis.
